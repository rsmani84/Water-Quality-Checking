# 💧 Water Quality Checker Virtual Lab

A simple Streamlit-based virtual lab to assess water quality using common physico-chemical parameters.

## Features
- Water quality testing based on:
  - pH
  - Turbidity
  - Hardness
  - TDS
  - Conductivity
- Water quality score calculation
- Safe / Needs Treatment / Unsafe classification
- Downloadable PDF report
- Sample dataset download
- Student feedback and rating section

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
